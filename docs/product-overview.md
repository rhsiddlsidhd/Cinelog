# 프로젝트 개요 — Cinelog

## 서비스 정보

| 항목 | 내용 |
|------|------|
| 서비스명 | Cinelog |
| 설명 | TMDB 데이터를 활용해 영화·드라마·인물 정보를 탐색하는 웹 앱 |
| 목적 | 원하는 영화·드라마를 쉽게 찾고 관련 정보를 한 곳에서 확인 |
| 대상 사용자 | 영화·드라마 콘텐츠를 즐기는 일반 사용자 |
| 핵심 가치 | 깔끔한 UI로 콘텐츠 탐색 경험 제공 |

## 핵심 기능 목록

| 기능명 | 설명 | 우선순위 |
|--------|------|---------|
| 탐색 | 트렌딩·인기·평점순·장르 필터로 영화·드라마 콘텐츠 둘러보기 | 상 |
| 검색 | 키워드로 영화·TV·인물 통합 검색 | 상 |
| 상세 보기 | 콘텐츠 정보·출연진·트레일러·추천작·시청 가능 플랫폼 확인 | 상 |
