# 화면 명세서 — Cinelog

## 홈

**화면 목적:** 앱 진입 시 트렌딩·인기·평점순 콘텐츠를 한눈에 보여주는 메인 화면

**주요 UI 요소**

- 트렌딩 가로 스크롤 섹션 (영화·TV 통합, day/week 탭)
- 인기 영화 섹션
- 인기 TV 섹션
- 평점 상위 콘텐츠 섹션
- 상단 검색 진입 버튼
- 하단 탭 내비게이션 (홈 / 탐색 / 검색)

**연결 화면:** 영화 상세 (카드 클릭), TV 상세 (카드 클릭), 탐색, 검색 결과

---

## 탐색

**화면 목적:** 장르·필터 조건으로 영화·TV 콘텐츠를 직접 탐색하는 화면

**주요 UI 요소**

- 콘텐츠 타입 탭 (영화 / TV)
- 장르 태그 선택 (복수 선택)
- 정렬 옵션 (인기순 / 평점순 / 최신순)
- 결과 그리드 (무한 스크롤)
- 하단 탭 내비게이션

**연결 화면:** 영화 상세 (카드 클릭), TV 상세 (카드 클릭)

---

## 검색 결과

**화면 목적:** 키워드 입력 후 영화·TV·인물 통합 검색 결과를 표시하는 화면

**주요 UI 요소**

- 검색 입력창 (자동 디바운스 적용)
- 결과 타입 탭 (전체 / 영화 / TV / 인물)
- 결과 목록 (포스터 썸네일 + 제목 + 부가 정보)
- 결과 없음 상태 처리
- 하단 탭 내비게이션

**연결 화면:** 영화 상세, TV 상세, 인물 상세 (결과 클릭)

---

## 영화 상세

**화면 목적:** 선택한 영화의 상세 정보를 보여주는 화면

**주요 UI 요소**

- 배경 이미지 (backdrop) + 포스터 오버레이
- 기본 정보 (제목 / 장르 / 개봉일 / 러닝타임 / 평점)
- 트레일러 재생 버튼 (YouTube 연결)
- 출연진 가로 스크롤 (배우명 + 배역명)
- 시청 가능 플랫폼 (OTT 아이콘)
- 추천 영화 가로 스크롤
- 유사 영화 가로 스크롤

**연결 화면:** 인물 상세 (출연진 클릭), 영화 상세 (추천·유사 영화 클릭)

---

## TV 상세

**화면 목적:** 선택한 TV 시리즈의 상세 정보를 보여주는 화면

**주요 UI 요소**

- 배경 이미지 (backdrop) + 포스터 오버레이
- 기본 정보 (제목 / 장르 / 첫 방영일 / 평점 / 시즌 수)
- 시즌 선택 드롭다운 + 에피소드 목록
- 트레일러 재생 버튼
- 출연진 가로 스크롤
- 시청 가능 플랫폼
- 추천 시리즈 가로 스크롤

**연결 화면:** 인물 상세 (출연진 클릭), TV 상세 (추천 시리즈 클릭)

---

## 인물 상세

**화면 목적:** 배우·감독 등 인물의 프로필과 필모그래피를 보여주는 화면

**주요 UI 요소**

- 프로필 이미지
- 기본 정보 (이름 / 직업 / 생년월일 / 출생지)
- 바이오그래피 (펼침/접힘)
- 출연 영화 목록 가로 스크롤
- 출연 TV 목록 가로 스크롤

**연결 화면:** 영화 상세 (출연작 클릭), TV 상세 (출연작 클릭)
